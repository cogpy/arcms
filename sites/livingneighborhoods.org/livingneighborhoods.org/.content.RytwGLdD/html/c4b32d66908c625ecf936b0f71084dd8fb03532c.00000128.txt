//  (A button will appear only if it is specified in an
// arrangement section as well as in an OBJECT section.)
//  The vertical and horizontal subsections produce nested
// arrangements, with the effect of nested single column
// and single row tables.
//   Numbers followed by 'px' specify the gap between
// objects, in pixels.  (This is a vertical or horizontal
// gap for vertical or horizontal specifications,
// respectively.) If no gap is specified the default gap
// associated with the panel is used.

// GROUPs give properties held in common by a number of

//  Currently all of the buttons and choices
// (for choosing pen color, etc.)

// Thus they will all have the following properties (except
// where other values are specified for particular buttons).
