An Introduction to A Pattern Language, the Book

253 patterns for towns, buildings, and construction

On this part of the web site you have direct access to every page of the famous book, first published in 1977. Today, 36 years later, sales of this book are still increasing every year: it is this book which gives our web site its name.

Back in 1977, the book first introduced the concept of people designing buildings for themselves, and guaranteeing the comfort and functionality of the buildings they designed, because the elements of the language are "patterns", elements which are a collective memory of things which work in our surroundings.

These tools allow anyone, and any group of people, to create beautiful, functional, meaningful places.
You can create a living world.
